Adding a EULA form to the VB6 Package and Deployment Wizard

#1: BACK UP YOUR C:\Program Files\Microsoft Visual Studio\VB98\Wizards\PDWizard\Setup1
      and don't forget to do #8

#2: Load the Setup1.vbp project into Visual Basic.

#3: Add the following to the project.
       Project  .... add the ritchtextbox to the toolbox
       Module ..... resfile.bas
       Form    ..... EulaForm1.frm ( if you load this before adding a Ritchtextbox it will error)

#4: Remove setup1.frm from the project and add the new setup1.frm
      or you can just edit the following ..

      In the setup1.frm Form Load event look for the following ...

            ' Display the background "blue-wash" setup screen as soon as we get the title
            '
            ShowMainForm
            '
            ' Display the welcome dialog
            '
            ShowWelcomeForm
            '
            'show are Eula form
            FrmEula.Show vbModal
#5: Open the setup1.res 



#6: Add a Custom Resource and locate your eula.rtf  file and add it.


#7: Save the Project and compile it, close the IDE.

#8: IMPORTANT STEP .....
      Since the setup1.exe is now using a ritchtextbox we need to have the bootstrap from setup.exe install it.
      To do this you need to do the following.

    A:  Run the PWD on your project and create the install. dont forget to add the ritchtextbox.ocx and dll.
          ( Riched32.dll and Richtx32.Ocx ) if your project doesn't use one.

    B: in the package folder locate the Setup.lst file and open it in Notepad.
        at the end of the [Bootstrap Files] section add the following information.
        ( this can be found in the [Setup1 Files] section of the file )

       File8=@RICHED32.DLL,$(WinSysPathSysFile),,,5/7/98 12:00:00 AM,174352,4.0.993.4
       File9=@RICHTX32.OCX,$(WinSysPath),$(DLLSelfRegister),$(Shared),5/22/00 12:00:00 AM,203976,6.0.88.4
